# CogniCare-Contributions

Contributed to the development of CogniCare (team project), an app that helps caretakers monitor dementia patients remotely

## Implemented Features

- Notification service - Enables patients to receive reminders as notifications and notifies caretakers when abnormal activity is detected (e.g. missed reminders, unusual heart rate)
- Location-tracking service - Allows caretakers to know patient's current location at all times
- Geofence service - Notifies caretakers if patients leave a certain distance from their designated "safe zone" 
- Patient home page - Added reminder pop-up with snooze functionality
- Location map page - Displays map of patient's current location

## Software Used
- Android Studio
- Dart
- Flutter
- Firebase
